# 资料场景方式选择 PRD

## 1. 页面范围

### 1.1 当前页面
- 页面名称：资料场景方式选择
- 所属模块：识别作业资料
- 页面路由/组件：/homework → 识别作业资料 → 资料场景方式选择步骤
- 相关代码文件：
- `src/components/UploadQuestionDialog.tsx`
- `src/app/homework/page.tsx`

### 1.2 上级页面/上游入口
- 上级页面：`/homework` 作业管理页中的上传录题弹窗。
- 入口位置：识别作业资料弹窗提交成功后进入，也可从后续步骤条返回。
- 进入方式：系统在收到资料、学科和识别范围后打开上传录题弹窗并展示模式选择。
- 传入数据：接收已上传文件、学段学科、识别范围和可选文件角色草稿。
- 依赖状态：必须已有待识别资料；从后续步骤返回时可能需要先确认清空当前识别进度。

### 1.3 下级页面/下游流程
- 下级页面/下游流程：框选识别步骤。
- 触发方式：用户选择「仅识别题目」或「识别题目和答案」；多文件题目答案模式下需先确认文件用途。
- 传出数据：传出识别模式和文件用途分配结果。
- 后续状态：进入框选识别步骤，开始展示资料预览和切题框选能力。
- 返回逻辑：后续步骤点击步骤条第一步时，确认后清空当前识别进度并回到本步骤。

### 1.4 范围边界摘要
- 本文档覆盖：记录资料场景方式选择页面的完整页面逻辑，覆盖模式选择卡片、已上传文件列表、文件角色分配弹窗、返回确认等交互边界。
- 本文档不覆盖：示例图片 alt 文案、多文件提示文案

## 2. 功能概述

记录资料场景方式选择页面的完整页面逻辑，覆盖模式选择卡片、已上传文件列表、文件角色分配弹窗、返回确认等交互边界。本文档按单页面/单流程沉淀，业务逻辑、验收标准和来源追溯均来自需求注册表。

## 3. 业务逻辑说明

### SELECT_MODE-001 页面标题与引导文案

来源：代码事实 + 确认决策

#### 显示说明

页面居中显示标题「请选择识别方式」，标题下方显示引导文案「建议根据您的资料内容，选择合适的处理流程」。

#### 验收标准

2. 引导文案正确显示

### SELECT_MODE-002 仅识别题目模式卡片

来源：代码事实 + 确认决策

#### 显示说明

左列显示「仅识别题目」卡片，包含蓝色文件图标、标题「仅识别题目」、副文案「适合只包含题目，不包含答案的资料内容」和一张示例图。

#### 操作说明

1、点击「仅识别题目」卡片后，进入一步识别流程，直接进入框选与识别页面。
2、如果上传了多个文件，所有文件均视为题目文件处理。

- 后续流程：将用户选择的「仅识别题目」模式传递给后续识别流程

#### 验收标准

1. 点击卡片后进入框选识别页面
2. 识别模式为仅识别题目
3. 多文件时不弹出角色分配弹窗

### SELECT_MODE-003 识别题目和答案模式卡片

来源：代码事实 + 确认决策

#### 显示说明

右列显示「识别题目和答案」卡片，包含紫色图层图标、标题「识别题目和答案」、副文案「适合包含即题目，也包含答案或解析的资料内容」和两张示例图。

#### 操作说明

1、点击「识别题目和答案」卡片后，进入分步识别流程。
2、如果上传了多个文件（≥2个），弹出「指定文件用途」弹窗，让用户为每份文件分配角色（题目文件/答案文件）。
3、如果只有 1 个文件，不弹窗，直接进入框选识别页面，默认该文件同时包含题目和答案。

- 后续流程：将用户选择的「识别题目和答案」模式和文件角色分配结果传递给后续识别流程

- 异常边界：关闭角色分配弹窗时，已选的识别模式取消，用户需重新选择模式

#### 验收标准

1. 点击卡片后进入分步识别流程
2. 多文件时弹出角色分配弹窗
3. 单文件时直接进入框选识别页面
4. 关闭弹窗后识别模式取消，可重新选择

### SELECT_MODE-004 已上传文件列表

来源：代码事实 + 确认决策

#### 显示说明

模式卡片下方显示「已上传的文件：」标题和文件列表。每行显示文件图标、文件名和识别页数（如「识别12页」）。页数信息来自上传弹窗中用户选择的识别范围。如果未获取到页数信息，则不显示页数。

#### 操作说明

文件列表为纯展示区域，用户不可在此处操作文件。

- 后续流程：文件数据来自用户上传时确定的文件和识别范围

#### 验收标准

1. 文件列表正确显示文件名
2. 有识别范围时显示识别页数
3. 无识别范围时不显示页数

### SELECT_MODE-005 文件角色分配弹窗

来源：代码事实 + 确认决策

#### 显示说明

弹窗标题「指定文件用途」，说明文案「检测到您上传了 N 个文件，请分别指定用途」。每份文件一行，显示文件图标、文件名、题目文件按钮、答案文件按钮。底部有取消和确认按钮。

#### 操作说明

1、每份文件的角色选择独立，A 文件选题目不影响 B 文件也选题目。
2、点击已选中的角色按钮可取消选择，恢复为未分配状态。
3、所有文件都分配角色后，「确认，开始处理」按钮高亮可点击。
4、点击「确认」后关闭弹窗，进入框选识别页面。
5、点击「取消」或右上角 ✕ 关闭弹窗，同时取消已选的识别模式，用户可重新选择模式。

- 后续流程：角色分配结果传递给后续流程，用于决定文件显示顺序和处理方式

- 异常边界：关闭弹窗前如果有文件未分配角色，直接关闭，不提示（用户可下次重新进入弹窗分配）

#### 验收标准

1. 每份文件独立选择角色
2. 点击已选中按钮可取消
3. 所有文件分配后确认按钮可用
4. 取消和 ✕ 都可使识别模式取消，返回重新选择

### SELECT_MODE-006 返回模式选择确认

来源：代码事实 + 确认决策

#### 显示说明

步骤条中第一步「资料场景方式选择」在后续步骤时为可点击按钮（带 ← 图标）。在模式选择步骤时则为当前选中态高亮。

#### 操作说明

1、在后续步骤中点击步骤条第一步，弹出确认弹窗：「返回模式选择将清空当前识别进度（包括框选和已识别的题目），是否继续？」。
2、点击「确认返回」后：清空当前识别模式和文件角色分配、已框选的题目区域、已识别的题目和答案数据，返回模式选择页面。
3、点击「取消」关闭弹窗，保持当前页面不变。

- 后续流程：确认后清空所有流程数据，用户需重新选择模式开始

- 异常边界：如果当前没有识别进度，无需清空，直接返回

#### 验收标准

1. 点击返回按钮弹出确认弹窗
2. 确认后清空所有数据返回模式选择页面
3. 取消后保持当前页面不变

### SELECT_MODE-007 步骤条在模式选择页的显示

来源：代码事实 + 确认决策

#### 显示说明

模式选择页面顶部显示 4 步步骤条：1.资料场景方式选择 → 2.框选&识别 → 3.核查题目信息 → 4.加入试卷。第一步「资料场景方式选择」为当前激活态（绿色高亮 + 脉冲圆点），其余三步为灰色未到达态。

#### 操作说明

步骤条在模式选择步骤仅展示流程阶段，不可点击切换步骤。

#### 验收标准

1. 步骤条正确显示
2. 第一步为当前激活态
3. 其余步骤为不可点击的灰色态
