# 原型使用说明

本文档从"识别模式"出发，按流程步骤描述所有可触发的页面、弹窗和交互分支，确保测试和演示时不会遗漏任何路径。

---

## 模式一：仅识别题目

适用场景：上传的资料只包含题目、没有答案或解析。

### 完整主流程

```
主页（AI小乐面板）
  → 点击「识别作业资料」
  → 识别作业资料弹窗（上传文件）
  → 点击「开始处理」
  → 步骤1「选择识别方式」
  → 点击「仅识别题目」
  → 步骤2「选择识别内容」
  → 系统自动切题（腾讯云）→ 框选框出现
  → 选中要识别的框 → 点击「开始识别」
  → 识别中（首次为全屏加载动画，含暂停/取消按钮）
  → 步骤3「核对识别结果」
  → 核对题目内容 → 点击「加入试卷」
  → 跳转到试卷编辑页面
```

### 步骤1 可触发的交互

| 操作 | 触发结果 |
|------|---------|
| 上传 1 个文件 | 直接进入步骤1，显示文件列表 |
| 上传 ≥2 个文件 | 进入步骤1，显示文件列表，不弹窗 |
| 点击「仅识别题目」卡片 | 进入步骤2 |
| 在步骤1页面点击步骤条 | 步骤条当前激活态，不可点击 |

### 步骤2 可触发的交互

| 操作 | 前提条件 | 触发结果 |
|------|---------|---------|
| 系统自动切题 | 进入步骤2时自动触发 | 框选框出现，默认全选 |
| 手动框选 | 鼠标按住拖动 | 新增一个未识别框 |
| 点击「全选」 | 有框时可用 | 全选/取消全选 |
| 点击「一键清空」 | 有框时可用 | 弹窗确认后删除全部框 |
| 点击选框复选框 | — | 切换单个框的选中状态 |
| 拖拽选框 | — | 移动框的位置 |
| 拖拽选框手柄 | — | 调整框的大小 |
| 点击「×」删除单框 | — | 删除单个框 |
| 点击「开始识别」 | 有框选中 | 进入识别流程（首次全屏加载，增量时进度条） |
| 点击「继续上传」 | — | 打开上传文件弹窗，追加文件后回到步骤2，仅对新文件切题 |
| 点击「重新上传」 | — | 弹出确认弹窗→确认后清空→回到上传文件弹窗→回到步骤1 |

### 步骤3 可触发的交互

| 操作 | 前提条件 | 触发结果 |
|------|---------|---------|
| 右侧查看题目卡片 | 识别完成后 | 显示题干内容（不含答案/解析区域） |
| 切换「图片模式」 | 有题目 | 查看题目原始截图 |
| 切换「编辑模式」 | 有题目 | 查看/编辑AI识别的文字内容 |
| 修改题型 | 有题目 | 切换题型下拉框 |
| 点击题号 | 有题目 | 右侧滚动定位到对应题目 |
| 使用图片裁剪 | 开启「使用图片」 | 裁剪题目图片 |
| 添加子题「+子题」 | 复合题型 | 新增空子题 |
| 点击步骤条第2步返回 | 在步骤3 | 回到步骤2，框保留但不选中，已识别题目保留 |
| 点击步骤条第1步返回 | 在步骤3 | 弹窗确认→清空→回到步骤1 |
| 点击「加入试卷」 | 有题目 | 如有缺口弹窗确认→跳转试卷编辑页 |

## 模式二：识别题目和答案

适用场景：上传的资料同时包含题目和答案/解析。

### 完整主流程

```
主页（AI小乐面板）
  → 点击「识别作业资料」
  → 识别作业资料弹窗（上传文件）
  → 点击「开始处理」
  → 步骤1「选择识别方式」
  → 点击「识别题目和答案」
  → [多文件时] 指定文件用途弹窗 → 分配角色 → 确认
  → 步骤2「选择识别内容」
  → 系统自动切题（腾讯云）→ 框选框出现
  → 选中要识别的框 → 点击「开始识别」
  → 识别中（首次为全屏加载动画，含暂停/取消按钮）
  → 步骤3「核对识别结果」
  → 核对题目与答案匹配 → 未匹配的手动关联 → 点击「加入试卷」
  → 跳转到试卷编辑页面
```

### 步骤1 可触发的交互

| 操作 | 前提条件 | 触发结果 |
|------|---------|---------|
| 上传 1 个文件 | — | 直接进入步骤1 |
| 上传 ≥2 个文件 | — | 进入步骤1 |
| 点击「识别题目和答案」卡片 | 仅 1 个文件 | 直接进入步骤2 |
| 点击「识别题目和答案」卡片 | ≥2 个文件 | 弹出「指定文件用途」弹窗 |
| 指定文件用途弹窗内操作 | 弹窗已显示 | 每份文件独立选「题目文件」/「答案文件」，全部分配后「确认」可用 |
| 文件用途弹窗点「取消」或「×」 | — | 关闭弹窗，识别模式取消，回到步骤1 |

### 步骤2 可触发的交互

| 操作 | 前提条件 | 触发结果 |
|------|---------|---------|
| （与仅识别题目模式的步骤2 相同） | — | — |
| 点击「继续上传」| — | 打开上传文件弹窗 |
| 继续上传后 | 新文件 + 已有文件已分配角色 | 弹出「指定文件用途」弹窗，仅显示新文件 |
| 继续上传后 | 之前只有 1 个文件从未分配角色 | 弹出「指定文件用途」弹窗，显示全部文件 |
| 点击「重新上传」 | — | 确认弹窗→清空→回到上传文件弹窗→回到步骤1 |
| 文件标签切换 | 有题目文件+答案文件 | 左侧预览区可切换查看题目文件/答案文件 |
| 「全局匹配答案」按钮 | 已有题目 + 未匹配答案 | 从整页资料自动匹配答案到已有题目 |

### 步骤3 可触发的交互

| 操作 | 前提条件 | 触发结果 |
|------|---------|---------|
| 右侧查看题目卡片 | 识别完成后 | 显示题干+答案/解析区域 |
| 匹配状态提示条 | 有未匹配的题目 | 显示「已匹配X题，待匹配Y题」 |
| 框选答案后关联弹窗 | 步骤3左侧框选答案区域 | 弹出「请关联题号」弹窗，选择题号后点击「关联」 |
| 手动编辑答案/解析 | 编辑模式 | 直接修改答案和解析文字 |
| 答案/解析自动拆分 | 解答题有子题标记 | 答案按子题标记拆分到各子题 |
| 删除已有内容的子题 | 点删除按钮 | 弹窗确认后一并删除 |
| 「全局匹配答案」按钮 | 框选识别阶段已有题目 | 自动从整页匹配答案 |
| （其余与仅识别题目模式的步骤3 相同） | — | — |

## 继续上传的跨模式行为

| 当前模式 | 行为 |
|---------|------|
| 仅识别题目 | 追加文件→直接切题→不弹窗 |
| 识别题目和答案（原 1 个文件） | 追加文件→弹窗显示全部文件→分配文件用途→切题 |
| 识别题目和答案（原 ≥2 个文件，已分配） | 追加文件→弹窗仅显示新文件→分配文件用途→切题 |

---

## 重新上传的跨模式行为

| 当前模式 | 行为 |
|---------|------|
| 仅识别题目 | 确认弹窗→清空所有数据→回到上传文件弹窗→回到步骤1 |
| 识别题目和答案 | 同上 |

---

## 增量识别的三种场景

| 场景 | 描述 | 进度条文案 | 题目卡片表现 |
|------|------|-----------|------------|
| A | 仅新增框选 | 「正在识别 X 个新增区域...」 | 已有卡片正常显示 |
| B | 仅重框已识别题目 | 「正在重新识别 X 道题目...」 | 重识别卡片置灰+「识别中」遮罩 |
| C | 混合新增+重框 | 「正在识别 X 个区域（含 Y 道重新识别）...」 | 重识别卡片置灰，其余正常 |

## 弹窗一览

| 弹窗名称 | 触发条件 | 所属步骤 |
|---------|---------|---------|
| 识别作业资料弹窗 | 主页点「识别作业资料」或点「继续上传」 | 步骤1前 |
| 多文件范围选择弹窗 | 上传文件总页数超过24页 | 步骤1前 |
| 指定文件用途弹窗 | 多文件+选择「识别题目和答案」 | 步骤1→步骤2之间 / 步骤2继续上传后 |
| 一键清空确认弹窗 | 步骤2点「一键清空」 | 步骤2 |
| 重新上传确认弹窗 | 步骤2或步骤3点「重新上传」 | 步骤2/步骤3 |
| 返回模式选择确认弹窗 | 步骤条回退到步骤1 | 步骤2/步骤3 |
| 请关联题号弹窗 | 步骤3框选答案区域 | 步骤3 |
| 删除子题确认弹窗 | 步骤3删除已有内容的子题 | 步骤3 |
| 加入试卷确认弹窗 | 步骤3点「加入试卷」且有缺口 | 步骤3 |
| 框类型选择弹窗 | 步骤2画框结束 / 步骤3画框结束 | 步骤2/步骤3 |
